## 学习建议

### 新手阶段（2-3周）
1. 理解大模型评测基本概念
2. 掌握常用评测指标
3. 学会使用evalScope基础功能

### 进阶阶段（3-6周）
1. 深入理解评测框架架构
2. 掌握评测任务设计
3. 完成模型对比评测项目

### 专家阶段（2个月以上）
1. 评测系统架构设计
2. 分布式评测优化
3. 前沿技术研究与应用
