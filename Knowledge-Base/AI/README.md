# 大模型评测知识库

## 概述

本知识库专为软件工程大四本科生设计，聚焦AI测试、大模型评测方向，提供从入门到专家的完整学习路径。知识库采用Obsidian风格和MOC（Map of Content）规范构建，确保知识结构清晰、层次分明。

## 知识库结构

```
AI/
├── README.md                           # 本文件
├── CardHome.md                         # MOC知识地图
├── 01-机器学习基础/                    # 机器学习基础知识模块
├── 02-深度学习与PyTorch/               # 深度学习与PyTorch框架
├── 03-Transformer框架/                 # Transformer架构与Transformers库
├── 04-LangChain开发/                   # LangChain框架开发
├── 05-大模型评测框架/                  # 大模型评测体系（侧重evalScope）
└── assets/                             # 资源文件目录
```

## 学习路径

### 快速入门路径（1-2周）
1. 机器学习基础概念
2. PyTorch基础操作
3. Transformer核心原理
4. 大模型评测基础

### 进阶路径（1-2个月）
1. 深度学习模型构建
2. LangChain应用开发
3. 评测框架深入
4. 实战项目练习

### 专家路径（3-6个月）
1. 模型优化与调优
2. 评测体系设计
3. 企业级实战
4. 前沿技术追踪

## 特色功能

- **结构化知识**：每个技术点都有基础、进阶、专家三个层次
- **实战导向**：包含大量代码示例和企业级demo
- **深度剖析**：对难点进行白话解释和类比说明
- **完整覆盖**：从理论到实践，从入门到专家

## 使用建议

1. 按顺序学习各模块内容
2. 结合代码示例动手实践
3. 关注"细节难点"和"常见疑问"部分
4. 定期复习CardHome.md中的知识地图
