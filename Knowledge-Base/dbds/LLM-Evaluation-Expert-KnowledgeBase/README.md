# LLM评测专家知识库

## 📚 知识库概述

本知识库专为软件工程大四本科生设计，聚焦AI测试、大模型评测方向，提供从入门到专家的完整学习路径。知识库采用Obsidian MOC规范构建，确保知识结构清晰、易于导航。

## 🏗️ 知识库架构

### 顶层结构
```
LLM-Evaluation-Expert-KnowledgeBase/
├── 01-机器学习基础/          # 机器学习核心概念与算法
├── 02-深度学习与PyTorch/     # 深度学习理论与PyTorch实战
├── 03-Transformer框架/       # Transformer原理与Transformers库
├── 04-LangChain开发/         # LangChain框架与RAG应用
├── 05-大模型评测框架/        # 大模型评测理论与实践
├── 06-实战项目/              # 企业级实战项目案例
├── 07-学习路线/              # 系统性学习路径规划
├── 08-工具与资源/            # 开发工具与学习资源
└── MOC.md                    # 主知识地图
```

### 知识卡片规范
- **文件命名**: 采用`序号-主题-子主题.md`格式
- **内容结构**: 由浅入深，包含基础概念、实战demo、进阶技巧
- **代码规范**: 遵循ipynb风格，高注释率(≥80%)
- **图示说明**: 使用Mermaid图表辅助理解

## 🎓 学习目标

### 短期目标（1-3个月）
- 掌握机器学习基础概念和常用算法
- 理解深度学习基本原理和PyTorch使用
- 熟悉Transformer架构和Transformers库

### 中期目标（3-6个月）
- 掌握LangChain框架开发能力
- 熟悉大模型评测框架evalScope
- 完成1-2个实战项目

### 长期目标（6-12个月）
- 具备企业级AI测试开发能力
- 能够独立完成大模型评测任务
- 达到AI测试工程师岗位要求

## 🚀 快速开始

### 学习顺序建议
1. **基础阶段**: 01 → 02 → 03
2. **应用阶段**: 04 → 05
3. **实战阶段**: 06 → 07
4. **拓展阶段**: 08

### 学习建议
- 按顺序学习，确保基础扎实
- 边学边练，完成每个模块的实战demo
- 定期复习，构建知识体系
- 参与社区，交流学习心得
